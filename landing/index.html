Getting Started

Three steps to full protection.

From zero to a live policy engine in under 5 minutes. No infra to manage.

1

Install

Add the SDK to your project. Works with LangChain, AutoGen, CrewAI, or raw API calls.

$ npm install @agentguard/sdk

# or Python
$ pip install agentguard
2

Define Policies

Write declarative YAML policies — version-controlled, reviewed in PRs. Enforced outside the model.

id: my-policy
version: 1.0.0
rules:
  - id: block-external
    action: block
    match:
      tool: http_post
      param.destination:
        notIn: allowlist
default: allow
3

Wrap your agent. Every action evaluated before execution — sub-millisecond, no compromises.

});

The Problem

Your agents are in production.
Your security team has no idea what they're doing.

Traditional security tools see the API call. They don't understand why an agent made it — or whether it should have.

🕳️

Zero Runtime Visibility

Your SIEM logs that a Stripe API was called for $4,200. It can't tell you if that was legitimate — or the result of a prompt injection attack via a crafted support ticket.

🚫

No Policy Enforcement Layer

System prompts are not security controls. They're probabilistic suggestions the LLM can ignore, misinterpret, or be manipulated into bypassing. There's nothing between the agent's decision and the action it executes.

📋

Compliance Evidence Doesn't Exist

EU AI Act Article 12 requires documented logging of every high-risk AI decision. When your auditor asks for it, "we have a system prompt in Notion" is not a compliance programme.

"I have 200 agents deployed across 6 departments. My board asks about AI risk every quarter and I have nothing to show them." — CISO, Financial Services Enterprise (discovery interview, Jan 2026)

Python 
import requests

decision = requests.post(
    "https://api.agentguard.tech/api/v1/evaluate",
    headers={"X-API-Key": "ag_live_..."},
    json={
        "tool": "http_post",
        "params": {"destination": url}
    }
).json()

if decision["result"] == "block":
    raise PolicyViolation(decision["reason"])
TypeScript / Node.js 
const res = await fetch(
  "https://api.agentguard.tech/api/v1/evaluate",
  {
    method: "POST",
    headers: {
      "Content-Type": "application/json",

    body: JSON.stringify({
      tool: "http_post",
      params: { destination: url }
    })
  }
);

if (result === "block") {
