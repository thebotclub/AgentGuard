## Listing Details

**Name:** AgentGuard

**Tagline:** Runtime security layer for AI agents — evaluate every tool call before execution

**Description:**
AgentGuard sits between your AI agent and its tools. Every tool call — database queries, HTTP requests, file operations, shell commands — is evaluated against configurable policies before execution.

Think of it as a firewall for AI agent actions.

🔴 Kill Switch — halt every agent in your tenant with one API call
🔍 Policy Engine — sub-millisecond in-process evaluation
🔗 Audit Trail — SHA-256 hash-chained, tamper-evident
🛡️ PII Detection — 9 entity types, detect/redact/mask
📦 Batch Evaluate — 50 tool calls in one request
🤝 Framework Integrations — LangChain, CrewAI, OpenAI, AutoGen, Vercel AI SDK, Express/Fastify

Free tier: 100K events/month. TypeScript and Python SDKs.
