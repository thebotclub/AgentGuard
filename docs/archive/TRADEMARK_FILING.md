# AgentGuard — Trademark Filing Guide

## Filing Authority
**IP Australia** — https://www.ipaustralia.gov.au/trade-marks

## What to File

### Mark
**AGENTGUARD** (word mark)

### Classes

**Class 9** — Computer software; downloadable software for monitoring, securing, and governing artificial intelligence agents; software development kits (SDKs) for AI agent security; software for policy enforcement and compliance automation.

**Class 42** — Software as a service (SaaS) featuring AI agent security and governance; cloud-based platform for runtime monitoring and policy enforcement of AI agents; computer security consultancy services; providing online non-downloadable software for AI compliance and audit.

## Cost Estimate
- 2 classes × AU$330 = **AU$660** (online filing via TM Headstart)
- TM Headstart (recommended): AU$200 preliminary assessment + AU$330/class
- Total via Headstart: **~AU$860**

## Filing Options

### Option 1: TM Headstart (Recommended)
1. Go to https://www.ipaustralia.gov.au/trade-marks/applying/tm-headstart
2. Pay AU$200 for preliminary assessment (2-5 business days)
3. IP Australia checks for conflicts
4. If clear, convert to full application (AU$330/class)
5. **Advantage**: Get feedback before committing full fee

### Option 2: Direct Filing
1. Go to https://www.ipaustralia.gov.au/trade-marks/applying
2. File directly — AU$330/class
3. Examination in 4-8 weeks
4. **Risk**: No early feedback on conflicts

## Priority: File BEFORE Investor Meetings
- Filing date establishes priority
- Investors expect IP protection
- Even "TM Headstart pending" signals professionalism

## Potential Conflicts to Check
- "AgentGuard" — search https://search.ipaustralia.gov.au/trademarks/search/quick before filing
- Also check WIPO Global Brand Database: https://branddb.wipo.int

## Timeline
1. **Week 1**: TM Headstart application (AU$200)
2. **Week 2-3**: Receive preliminary assessment
3. **Week 3**: Convert to full application (AU$660)
4. **Month 2-3**: Examination
5. **Month 3-4**: If no opposition, registration
