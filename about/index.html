Hani Kashi
Founder & CEO

Security is only meaningful when it's invisible to the developer and inescapable to the threat. That principle drove Hani to build AgentGuard after years working at the intersection of enterprise security architecture and AI systems.

The catalyst was straightforward: watching a production AI agent — deployed inside a financial services firm — execute a series of high-privilege operations that no human had explicitly approved, with no record that it had happened. The tools to prevent or detect that scenario simply didn't exist. So we built them.

AgentGuard is the runtime security layer that should have existed from the moment organisations started deploying autonomous agents. It imposes no opinion on how you build — it enforces the policies you define, logs everything cryptographically, and lets you halt an agent in milliseconds if something goes wrong.
